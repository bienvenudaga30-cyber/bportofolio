🚀 Déploiement sur GitHub Pages

Créez un dépôt GitHub nommé bportofolio (ou votrenom.github.io)

Uploadez tous les fichiers (respectez la structure ci-dessus)

Allez dans Settings → Pages

Source : Deploy from a branch → main → / (root)

Attendez ~2 minutes → votre site est en ligne !

URL de votre site :https://bienvenudaga30-cyber.github.io/bportofolio/

🚀 Déploiement sur Netlify (alternative)

Glissez-déposez le dossier du projet sur netlify.com/drop

Votre site est immédiatement en ligne avec une URL personnalisée
